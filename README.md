# Power-Explorer-landing

A landing page for Power Explorer

## Requeriments

  * Node 6+



## To install

  ```  bash
  npm install
  ```
  
## Developing

Unleash server (includes watch task for js and scss files):
  
  ```  bash
  npm start
  ```
  
For build JS files:

  ```  bash
  npm run build
  ```

Go to [http://localhost:5000](http://localhost:5000 "localhost") and have fun! :grin:

## Deploying

Deploy to gh-pages:
  
  ```bash
  npm run publish
  ```
